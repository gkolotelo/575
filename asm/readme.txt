Developed by:

Usage: python assembler.py [input_file.es575] [output_file(.bin/.mif)] [options]
Options:
    -h --help : Help file
    -m --mif  : Output Memory Initialization File format (.mif)

output_file must be indicated! Tested on Python 2.7.x
